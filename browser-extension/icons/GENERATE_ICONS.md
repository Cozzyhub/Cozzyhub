# Generate Extension Icons

You need to create 3 icon files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Quick Methods

### Method 1: Use Favicon Generator (Easiest)
1. Go to https://favicon.io/favicon-generator/
2. Create a simple design (e.g., "M" letter for Meesho)
3. Download the generated package
4. Rename and save the appropriate sizes here

### Method 2: Use AI Image Generator
Ask ChatGPT/DALL-E:
> "Create a simple, modern icon for a shopping extension. Purple and pink gradient, shopping bag symbol. 128x128 pixels."

Then resize for other dimensions.

### Method 3: Use Canva
1. Go to https://www.canva.com
2. Create custom dimensions (128x128)
3. Design a simple icon
4. Download as PNG
5. Resize for other dimensions using any image editor

### Method 4: Quick Placeholder (For Testing)
Create simple colored squares using any image editor:
- 16x16 purple square → `icon16.png`
- 48x48 purple square → `icon48.png`
- 128x128 purple square → `icon128.png`

## Design Suggestions

**Colors:** Purple (#667eea) and Pink (#764ba2) - matches Cozzyhub theme
**Symbol Ideas:**
- Shopping bag icon
- Letter "M" for Meesho
- Cart with plus sign
- Import/download arrow

**Keep it simple!** The icon will be small in the browser toolbar.
